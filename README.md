## Live NBA Scores API
Welcome to this cool API I built to pull live NBA scores from the ESPN website!

## Getting Started ##
First, clone this repository:

### `git clone https://github.com/BorisB93/nba-api.git`

Install project's python dependencies:

### `pip install -r requirements.txt`

Make sure you've also installed the frontend! Read the README at:

### `https://github.com/BorisB93/nba-app.git`

## Run!
In your preferred CLI, run the scheduled scraping script and the API:

### `py scheduledscraper.py`
### `py api.py `

## How about a test?
In your preferred CLI, run:
### `py tests/test_scraper.py `
